JavaScript - Deferred call imitation

Execute the following code:

    for (var i = 1; i <= 5; i++) {
      setTimeout(function () {
        console.log(i);
      }, 0);
    }
Explain the results.
        
    
    V1:
    for (var i = 1; i <= 5; i++) {
      setTimeout(function () {
        console.log(i); //this method will be called later and refer to the 'i' variable wich returns the '6' value at that moment
      }, 0);
    }
    

    

    

    
    V2:
    for (var i = 1; i <= 5; i++) {
      const currentIndex = i;
      setTimeout(function () {
        console.log(currentIndex);
      }, 0);
    }
