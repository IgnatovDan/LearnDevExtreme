DevExtreme - Detailed chart data

Display product categories in dxChart (bar chart). Argument is a category name, the value is product count that associated with the category. Data can be downloaded from http://services.odata.org/V3/Northwind/Northwind.svc/.  When a bar is clicked, display dxPopup with dxDataGrid inside. dxDataGrid should display products associated with the current category.

a.Knockout approach
b.jQuery approach
c.Angular approach

It is possible to get product count associated with each category using the Products navigation property. See http://services.odata.org/V3/Northwind/Northwind.svc/$metadata.
